Bienvenido a mi Página HTML


HTML Es el lenguaje con el que se escriben páginas web. Las páginas web pueden ser vistas por el usuario mediante un tipo de aplicación llamada navegador

Este lenguaje nos permite aglutinar textos, sonidos e imágenes y combinarlos a nuestros gusto.

El HTML nos permite la introducción de referencias a otras páginas por medios de los enlaces hipertexto.
